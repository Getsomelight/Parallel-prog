# Умножение квадратных матриц (C++ + Python верификация)

Программа на C++ умножает две квадратные матрицы, сохраняет результат в файл и выводит время выполнения.  
Скрипт на Python автоматизирует запуск, проверяет корректность умножения (через `numpy`) и собирает статистику времени для разных размеров матриц.

## Задача

- Перемножить две квадратные матрицы, заданные в текстовых файлах.
- Сохранить результат в файл.
- Вывести время выполнения.
- Автоматически верифицировать результат с помощью сторонней библиотеки (Python + NumPy).

## Ход работы

В ходе лабораторной работы были написаны файлы `lab1.cpp`, являющийся программой для перемножения квадратных матриц, и `solve_verification.py`, включающий в себя автоматический генератор матриц размерами 200, 400, .., 1800, 2000, автоматический запуск исполняемого файла `lab1.exe`, и проверку вычислений перемножения матриц.

Сгенерированные матрицы записывались в файлы `matrix_1.txt` и `matrix_2.txt` соответственно, затем запускалась программа `lab1.exe`, которая считывала данные файлы и записывала `result.txt`, выводила время, исходные матрицы и результирующую матрицу. Затем в `solve_verification.py` считывалось время перемножения матриц, и открывался файл с результирующей матрицей для проверки правильности решения.
