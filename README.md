# Умножение квадратных матриц (C++ + Python верификация)

Программа на C++ умножает две квадратные матрицы с распараллеливанием через OpenMP, сохраняет результат в файл и выводит время выполнения.  
Скрипт на Python автоматизирует запуск, проверяет корректность умножения (через `numpy`) и собирает статистику времени для разных размеров матриц и потоков. Количество ядер фиксированно (4).

## Задача

- Модернизировать 1 ЛР для работы с OpenMP

## Вывод по итогу ЛР

Распараллеливание через OpenMP в среднем выигрывает по времени, чем код без распараллеливания. Выборка по времени получалась из 10 запусков для каждого размера матриц и потоков (против 100 без распараллеливания, ради экономии времени), значения получились менее точными.
